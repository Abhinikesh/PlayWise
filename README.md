# PlayWise
PlayWise is a modern Android application designed to provide users with comprehensive information about various sports in an interactive and user-friendly way. The app combines learning, exploration, and engagement through features like rules, grounds, quizzes, and more.


🚀 Features

🏏 Multiple Sports Support
Explore sports like Cricket, Football, Basketball, Volleyball, Hockey, Tennis, Badminton, and Rugby.

📖 Detailed Rules Section
Well-structured rules with formatted text and easy readability.

🏟️ Ground Information
View multiple ground details for each sport with images and specifications.

🔍 Search Functionality
Quickly find any sport using the search bar.

🌙 Dark Mode Support
Toggle between light and dark themes for better user experience.

⭐ Favorites System
Save your favorite sports for quick access.



🧠 Interactive Quiz Section

Multiple choice questions

Score tracking

Random questions each time

Quiz history saved using database

📊 Score Tracking & History
View past quiz scores with performance insights.


🔊 Text-to-Speech (TTS)
Listen to sport rules with built-in voice reading.


📡 Live Score (Web Integration)
Redirects to live match scores via browser.


📤 Share Feature
Share sport rules easily with others.


🌐 Offline Mode Indicator
Displays internet connectivity status.


🛠️ Tech Stack

Language: Kotlin

UI: XML + Material Design

Architecture: MVVM

Database: Room Database

Storage: SharedPreferences

Other APIs: TextToSpeech, ConnectivityManager, Intents
